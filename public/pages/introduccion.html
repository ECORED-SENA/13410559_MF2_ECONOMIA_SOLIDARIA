Introducción

Una vez definida la economía solidaria y la importancia de este sector, en este material de formación se revisará el procedimiento para crear una empresa del sector solidario como las cooperativas. Adicionalmente, se estudiará su estructura funcional y las diferentes actividades que se deben desarrollar para constituir este tipo de organizaciones de acuerdo con la normatividad vigente.

Por consiguiente, para crear organizaciones del sector solidario, en este caso la cooperativa, es necesario definir los asociados, requerimientos documentales, estatutos, estructura administrativa, entre otros aspectos fundamentales que aprenderá con este módulo.
